
Beskrivning

Iliza AB är ett företag som erbjuder transport, service till skolor och uthyrning av bröllopslokaler.
Den här webbapplikationen visar företagets tjänster, personal och låter kunder boka service via ett enkelt formulär.

Webbsidan använder HTML, CSS, JavaScript och Bootstrap och innehåller funktioner som validering och datumkontroll.

Funktioner

Visa information om företaget och dess personal

Visa företagets tjänster med bilder och beskrivning

Formulär för servicebokning med:

Datum (tidigare datum går inte att välja)

Kundens namn

Meddelande

Statusmeddelande när bokning skickas

Responsiv design med Bootstrap
